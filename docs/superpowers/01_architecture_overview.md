## 3. и®ҫи®ЎеҺҹеҲҷ

1. **Plan-First**: LLM еҸӘиҫ“еҮә QueryPlan (з»“жһ„еҢ– JSON)пјҢдёҚеҒҡ SQL
2. **Validation-as-Gate**: жҜҸдёӘйҳ¶ж®өйғҪжңүдёҘж јйӘҢиҜҒпјҢеӨұиҙҘеҸҜйҮҚиҜ•
3. **Multi-Model**: еҗҢдёҖеҘ— Prompt + жөҒж°ҙзәҝйҖӮй…ҚдёҚеҗҢ LLM жҸҗдҫӣе•Ҷ
4. **Safety-by-Design**: зӯ–з•Ҙеј•ж“Һ + зұ»еһӢжЈҖжҹҘ + ж–№иЁҖзәҰжқҹ
5. **Observability**: еҶ…зҪ® OpenTelemetry + Prometheus жҢҮж Ү
